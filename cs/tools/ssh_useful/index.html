Generate a key

ssh-keygen -t rsa -b 4096 -C "your_email@gmail.com"

Give a proper name for the key, or else, it would overwrite the id_rsa file

Add a key

ssh-add -K ~/.ssh/id_rsa_my_key

For Mac, you need to add -k

# Use the following command to check the result
ssh-add -l
