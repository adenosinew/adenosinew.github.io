Python Platform module

platform

Access to underlying platform’s identifying data

# Get Architecture, The function relies on the system’s file command to do the actual work. 
print(platform.architecture())
# >> ('64bit', '')

# More reliable way to get the architecture of current interpreter
is_64bits = sys.maxsize > 2**32
# >> True

# Returns the machine type, e.g. 'i386'. An empty string is returned if the value cannot be determined.
print(platform.machine())
# >> x86_64

# Returns the computer’s network name (may not be fully qualified!). An empty string is returned if the value cannot be determined.
print(platform.node())
# >> Adenosine-MacBook-Pro.local

#
print(platform.platform())
# >> Darwin-18.2.0-x86_64-i386-64bit

# Returns a single string identifying the underlying platform with as much useful information as possible.
print(platform.processor())
# >> i386
