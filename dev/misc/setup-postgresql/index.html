How to Setup Postgresql

Installation

Ubuntu

sudo apt update
sudo apt install postgresql postgresql-contrib

Manjaro

pacman -S postgresql

Authentication

New installation has a postgres user by default

postgres has no DB password set on Ubuntu by default.

You can login to that account only by using the postgres OS user account

sudo -u postgres psql

Change the password

psql> ALTER USER <username> PASSWORD 'newpassword';

Rule of thumb:

YOU SHOULD NEVER EVER SET A PASSWORD FOR THE POSTGRES USER.

Allow remote connection

By default PostgreSQL is configured to be bound to “localhost”

Check the port (5432)

Run following command to see what is the port 5432 bound to

netstat -nlt

Proto Recv-Q Send-Q Local Address           Foreign Address         State
tcp        0      0 0.0.0.0:443             0.0.0.0:*               LISTEN        
tcp        0      0 0.0.0.0:80              0.0.0.0:*               LISTEN
tcp        0      0 0.0.0.0:22              0.0.0.0:*               LISTEN
tcp        0      0 127.0.0.1:5432          0.0.0.0:*               LISTEN

As we can see above port 5432 is bound to 127.0.0.1. It means any attempt to connect to the postgresql server from outside the machine will be refused. We can try hitting the port 5432 by using telnet.

telnet <remote-server-ip-address> 5432

Configuring postgresql.conf

Open /etc/postgresql/postgresql.conf

Replace the

listen_addresses = 'localhost'

with

listen_addresses = '*'

And restart the PostgreSQL service

Configuring pg_hba.conf

Open pg_hba.conf and add following entry at the very end

# First entry for IPv4 and second entry for IPv6
host    all             all              0.0.0.0/0                       md5
host    all             all              ::/0                            md5

Configuring firewall

In real world, you might have firewall running

Open port 5432 for TCP

More configuration

PostgreSQL can be configured and tuned through a series of configuration files.

Understanding postgresql.conf

Most global configuration settings are stored in postgresql.conf, which is created automatically when you install PostgreSQL.

Tune Authentication Options through pg_hba.conf

The pg_hba.conf file handles the default authentication options for client connections to the database.

PG CLI

Pgcli is a command line interface for Postgres with auto-completion and syntax highlighting.

Installation

macOS

brew install pgcli

Others

pip install pgcli

Usage

pgcli postgres://<username>:<passw0rd>@<host>:5432/<database>
